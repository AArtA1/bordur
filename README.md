# BORDUR

Статический сайт производителя тротуарной плитки, бордюров и компенсаторов.

## Технологии

Чистый HTML + CSS + JS. Без фреймворков, сборщиков и npm.

## Структура

```
bordur/
├── index.html                 ← Главная страница
├── css/style-v3.css           ← Все стили (дизайн-система)
├── js/
│   ├── header.js              ← Динамическая шапка + мобильное меню
│   ├── footer.js              ← Динамический футер + cookie-бар
│   ├── main-v3.js             ← Ядро JS (анимации, карта, формы)
│   ├── lead-form.js           ← Модальная форма заявки
│   ├── catalog-plitka.js      ← Фильтры каталога плитки
│   ├── catalog-bordyury.js    ← Фильтры каталога бордюров
│   └── catalog-kompensatory.js← Фильтры каталога компенсаторов
├── pages/                     ← Внутренние страницы
│   ├── about.html
│   ├── catalog.html
│   ├── catalog-plitka.html
│   ├── catalog-bordyury.html
│   └── catalog-kompensatory.html
├── img/                       ← Фото продуктов (plitka/, bordyury/, kompensatory/)
├── img-main/                  ← Фото для hero/галереи
├── video/production.mp4       ← Видео для баннера
└── data/                      ← Исходные данные о продуктах
```

## Запуск

Сайт статический — достаточно открыть файл в браузере или поднять любой локальный сервер.

### Вариант 1 — Python (встроен в macOS/Linux)

```bash
cd bordur
python3 -m http.server 8080
```

Открыть: [http://localhost:8080](http://localhost:8080)

### Вариант 2 — VS Code Live Server

Установить расширение **Live Server** → правой кнопкой на `index.html` → **Open with Live Server**.

### Вариант 3 — npx serve (если есть Node.js)

```bash
npx serve .
```

### Вариант 4 — открыть напрямую

Двойной клик на `index.html` — откроется в браузере как `file://`.
> Карта Яндекс и некоторые fetch-запросы могут не работать без сервера из-за CORS.

## Разработка

- Все стили — в `css/style-v3.css`, новые правила добавлять в конец файла
- Шапка и футер генерируются через JS (не трогать inline HTML в страницах)
- Карта Яндекс: заменить `YOUR_API_KEY` в `js/main-v3.js` на реальный ключ
- Методология CSS: BEM, акцентный цвет `--orange: #EA7627`

## Деплой

### Vercel (рекомендуется)

В репозитории уже есть `vercel.json` с нужными настройками.

**Через Vercel CLI:**

```bash
npm i -g vercel
cd bordur
vercel
```

Следовать инструкциям в терминале. При первом деплое спросит про настройки — всё оставить по умолчанию (framework: Other, build command: пусто, output directory: пусто).

**Через GitHub (автодеплой):**

1. Запушить репозиторий на GitHub
2. Зайти на [vercel.com](https://vercel.com) → **Add New Project**
3. Импортировать репозиторий
4. Настройки оставить пустыми (сборка не нужна) → **Deploy**

После деплоя каждый `git push` будет автоматически обновлять сайт.

**Переменные окружения:** не требуются (ключ Яндекс Карт вшит в `js/main-v3.js`).

### Другие хостинги

Любой статический хостинг: Nginx, Apache, GitHub Pages, Netlify.
Достаточно загрузить все файлы как есть — сборка не требуется.
